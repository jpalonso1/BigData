Project: Big Data Homework 2 (CVA)
Author: Juan Pablo Alonso

This program creates an artificial group of counterparties and runs a monte carlo simulation, given
then input parameters, to obtain a CVA value.

EXEC: CVA2
INPUT: 
-1 (OPT) Parameters file
OUTPUT: 
-Print to screen counterparty processing times, simulation times, and estimated CVA value.
